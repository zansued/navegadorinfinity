# Navegador Infinity AI

Gerenciador de múltiplos perfis Chromium com recursos anti-fingerprint, proxy individual por perfil e sincronização em nuvem via Supabase.

## 🚀 Recursos
- 👥 **Gerenciamento de Perfis**: Crie e isole múltiplos perfis do Chromium de forma simples.
- 🔒 **Privacidade Avançada**: Recursos para isolamento e proteção de impressões digitais de navegação (anti-fingerprints).
- 🌐 **Proxy por Perfil**: Configure proxy individualizado com suporte a autenticação por perfil.
- ☁️ **Sincronização em Nuvem**: Armazenamento e sincronização dinâmica de configurações de perfis usando banco de dados Supabase.
- 🎨 **Interface Neumórfica**: UI limpa e moderna utilizando Tkinter (Dark Theme).

## 📁 Estrutura do Projeto
- `manager/`: Script principal do gerenciador (`infinity_manager.py`) e arquivos de configuração.
- `extension/`: Extensão customizada do Chrome para integrar recursos de segurança e controle de logs de navegação.
- Scripts utilitários de distribuição e assinatura de pacotes PowerShell/BAT.
